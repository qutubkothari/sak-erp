## Why This Architecture?

### API (Production Mode) âœ…
- **Build works**: NestJS builds successfully on t2.micro
- **Small memory footprint**: ~30MB to build
- **Fast compilation**: ~30 seconds
- **Optimized**: Production-ready with all optimizations

### Web (Development Mode) âœ…
- **Production build FAILS**: Requires 4-6GB RAM, we only have 1GB
- **Build hangs**: Gets stuck at "Creating optimized production build"
- **Dev mode works perfectly**: 
  - No build required
  - Fast startup (~5 seconds)
  - All features functional
  - Low memory usage
  - PM2 disables file watching (no hot reload overhead)
